# Basketball 3D Pro - Debugging & Source Code

هذه النسخة تحتوي على الكود المصدري الكامل مع نظام تتبع أخطاء (Logging) متطور لحل مشكلة الشاشة السوداء العشوائية.

## كيفية تشغيل اللعبة
1. قم بفك ضغط الملف.
2. افتح مجلد `public`.
3. قم بتشغيل خادم محلي (Local Server). يمكنك استخدام:
   - **VS Code Live Server** (موصى به).
   - أو عبر Terminal: `python -m http.server 8000`.
4. افتح المتصفح على الرابط: `http://localhost:8000`.

## كيفية استخدام نظام الـ Logs (في حال حدوث شاشة سوداء)
إذا ظهرت لك الشاشة السوداء مرة أخرى، يرجى اتباع الخطوات التالية:
1. اضغط على مفتاح **"L"** في لوحة المفاتيح. ستظهر نافذة سوداء صغيرة في أسفل الشاشة تحتوي على سجل العمليات.
2. إذا لم تظهر النافذة، اضغط على **F12** لفتح "Console" المتصفح.
3. ابحث عن أي رسائل باللون الأحمر (Errors).
4. قم بتصوير الشاشة أو نسخ السجلات وإرسالها لي لأتمكن من معرفة أي جزء من الكود يفشل في جهازك تحديداً.

## التعديلات التقنية في هذه النسخة
- **Global Logger:** نظام يسجل كل خطوة من خطوات التحميل (Renderer, Camera, Physics, Court).
- **Error Boundaries:** حماية حلقة اللعبة من الانهيار الكامل في حال حدوث خطأ مفاجئ.
- **Race Condition Fix:** تم تعديل ترتيب التحميل لضمان عدم بدء التصيير إلا بعد جاهزية الحاوية (Canvas Container).
- **NBA Regulation Scaling:** تم ضبط أحجام الكرة والسلة لتكون متوافقة تماماً مع المقاييس العالمية.
